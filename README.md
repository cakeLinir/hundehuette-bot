# Hundehütte Bot

Der offizielle Community Discord Bot von **hundekuchenlive**.

## Setup

1. `npm install`
2. `.env` Datei erstellen mit `TOKEN`, `CLIENT_ID`, `GUILD_ID`
3. `npm run deploy` — Slash Commands registrieren
4. `npm start` — Bot starten

## Features

- Willkommensnachrichten & Rollen-Vergabe
- Ticket System
- Feedback System
- Auto Moderation
- Custom Commands
- Modulares System (Features pro Server an/aus)
- Mehrsprachigkeit (DE/EN)

## Tech Stack

- Node.js
- discord.js v14
- Slash Commands
